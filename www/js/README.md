# Cherry Web App

This is the web interface to the Cherry Ray Tracer.

## Build

This is needed to build the Wasm binary and JS glue.

```console
# Build for local development and testing
npm run build:dev

# Production build
npm run build:prod
```

## Develop

Start the development server:

```console
npm run start
```

Run the tests:

```console
npm run test
```
